Dynamic Typing

Variable creation

A variable, think of it as a name, is created when it is first assigned.

Future assignments only change the value associated with the name.

Variable types

There is no such thing. Types live with the objects the variable references, and not the variable itself.

Variable use

When a variable is used in an expression, it is immediately replaced with the object it currently refers to.

Variables are entries in a system table with space for links to objects.

Objects are pieces of allocated memory holding the values they represent.

References are pointers from variables to objects.

Objects are garbage-collected when the object reference counters are decremented to 0.
