Dictionaries

Dictionaries are collections of objects stored by key rather than position.

They are mutable and are not ordered.

Dictionary literals are curly braces with a series of key-value pairs:

>>> D = {'pi': 3.14, 'filling': 'cherry'}
>>> D['pi']
3.14
>>> D = dict(color='blue', age=42)
>>> D
{'color': 'blue', 'age': 42}

Initialize a dictionary using the keys. If no default value is given, the values are initialized to None:

>>> D = dict.fromkeys([1, 2], 0)
>>> D
{1: 0, 2: 0}

Unlike out-of-bound assignments in lists, assignments to new dictionary keys creates those keys:

>>> D = {}
>>> D['pi'] = 3.14
>>> D['filling'] = 'cherry'
>>> D
{'filling': 'cherry', 'pi': 3.14}

Keys, values, items

Show keys:

>>> D.keys()
['filling', 'pi']

Show values:

>>> D.values()
['cherry', 3.14]

Show keys and values. Shows them as a list of tuples:

>>> D.items()
[('filling', 'cherry'), ('pi', 3.14)]

A default value can be provided if a key is requested that does not exist:

>>> D.get('missing key', 42)
42

Sorting Keys

>>> for key in sorted(D):
...     print key
