Numbers

Some examples of numbers are integers, floating-point numbers, complex numbers, fixed-precision decimals, rational fractions, sets, and booleans.

Floating-point numbers are implemented as C doubles and have as much precision as the C compiler used in building the Python interpreter.

Hexadecimal digits start with a leading 0x or 0X.

Octal digits start with a leading 0o or 0O (leading zero followed by lower or upper-case O)

Binary literals begin with a leading 0B or 0b.

Integers can be converted to string with hex(I), oct(I) or bin(I).

Strings can be converted back to an integer with int(str, base).

Complex numbers can be created with complex(real, imaginary).

Comparison operators can be chained.

Mixed types are converted up to the most complicated operand, and then the expression is evaluated:

>>> 2 + 2.2
4.2

Floor versus Truncation

Floor reduces to the lowest integer, whereas truncate drops the fractional:

>>> import math
>>> math.floor(6.5)
6
>>> math.floor(-6.5)
-7
>>> math.trunc(6.5)
6
>>> math.trunc(-6.5)
-6

Random module

Random supplies a random floating-point number between 0 and 1:

>>> import random
>>> random.random()
0.5808514274117722
>>> random.randint(1, 10)
3
>>> random.choice([1, 2, 3])
2

Decimal

>>> import decimal
>>> decimal.Decimal('0.33333') + decimal.Decimal('0.44444')
Decimal('0.77777')
>>> decimal.getcontext().prec = 3
>>> decimal.Decimal('0.33333') + decimal.Decimal('0.44444')
Decimal('0.778')

Fraction Type

Implments a rational number object in which the numerator and denominator are kept explicitly.

>>> from fractions import Fraction
>>> x = Fraction(1, 5)
>>> y = Fraction(4, 10)
>>> x
Fraction(1, 5)
>>> y
Fraction(2, 5)
>>> print(y)
2/5
>>> (3.5).as_integer_ratio()
(7, 2)
