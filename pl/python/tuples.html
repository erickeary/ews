Tuples

Tuples are immutable sequences coded in parentheses.

Tuples can be used like a list except that immutability is desired.

Tuples can be nested.

Examples of tuples:

>>> T = (1, 2, 3)
>>> len(T)
3
>>> T = tuple('green')
>>> T
('g', 'r', 'e', 'e', 'n')

Concatenation:

>>> T + (4)
(1, 2, 3, 4)

One-item tuple. The comma is necessary to prevent it being evaluated as an expression.

>>> T = (42,)
