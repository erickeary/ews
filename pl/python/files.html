Files

There is no literal syntax for creating a file. A file may be created similar to the following: 

>>> F = open('example.txt', 'w')
>>> F.write('42')
>>> F.close()

A file can be opened for reading:

>>> F = open('example.txt')  # 'r' can be omitted because it is the default processing mode
>>> age = F.read()
>>> age
'42'

Note that the file contents are returned as a string regardless of the data type in the file.

Read the next line into a string (includes newline):

>>> a_string = input.readline()

Read all lines into a list of strings (includes newlines):

>>> a_list = input.readlines()

Write all line strings from a list to a file:

>>> output.writelines(a_list)

File iterators read a line at a time:

for line in open(r'C:\example.txt', 'w'):
    print(line)

Pickle

Native Python objects can be stored using pickle.

>>> D = {'key5': 5, 'key6': 6}
>>> F = open('example.pkl', 'wb')
>>> import pickle
>>> pickle.dump(D, F)
>>> F.close()

Use pickle to read them back in:

>>> F = open('example.pkl', 'rb')
>>> E = pickle.load(F)
>>> E
{'key5': 5, 'key6': 6}
