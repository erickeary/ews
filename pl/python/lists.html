Lists

Lists are positionally ordered collections of arbitrarily typed objects with no fixed size.

Lists are mutable.

Sequence Operations

Empty list:

>>> L = []

Index by position:

>>> L = ['String', 3.14, 42]
>>> L[2]
42

List length:

>>> len(L)
3

Slicing a list will return a new list:

>>> L[:-1]
['String',3.14]

Concatenating a list with another returns one new list:

>>> L + ['e',2.71]
['String', 3.14, 42, 'e', 2.71]

List Specific Operations

Adding to the end of a list:

>>> L.append(2.71)
>>> L  # lists are mutable and will show changes
['String', 3.14, 42, 2.71]
>>> L.extend(['4', 4])
>>> L
['String', 3.14, 42, 2.71, '4', 4]
>>> L.insert(2, 'day')
>>> L
['String', 3.14, 'day', 42, 2.71, '4', 4]

Position of a value:

>>> L.index(42)
3

Count how many times a value appears:

>>> L.count('day')
1

Sorting:

>>> L.sort()
>>> L
[2.71, 3.14, 4, 42, '4', 'String', 'day']

Reverse a list:

>>> L.reverse()
>>> L
['day', 'String', '4', 42, 4, 3.14, 2.71]

Removing an item:

>>> L.pop(2)  # could also do 'del L[2]'
42
>>> L
['String', 3.14, 2.71]

Although lists are not a fixed size, referencing beyond the bounds will produce an error.

Lists can be nested, e.g., to form a matrix or "multidimensional array".

>>> M = [[9, 8, 7],
         [6, 5, 4],
         [3, 2, 1]]
>>> M
[[9,8,7],[6,5,4],[3,2,1]]
>>> M[2][0]
3

Comprehensions

List comprehensions build a new list by running an expression on each item in a sequence from left to right.

>>> col1 = [row[0] for row in M]
>>> col1
[9, 6, 3]

List comprehensions are enclosed in square brackets (the list part), and are composed of an expression and a looping construct that share a variable.

Comprehensions can be combined with if statements for filtering:

>>> [row[1] for row in M if row[1] % 2 == 0]  # filter out odd
[8, 2]

Works for literal lists and strings, too:

>>> diag = [M[i][i] for i in [0, 1, 2]]
>>> diag
[9, 5, 1]
>>> duplicates = [c * 3 for c in 'String']
>>> duplicates
['SSS', 'ttt', 'rrr', 'iii', 'nnn', 'ggg']
