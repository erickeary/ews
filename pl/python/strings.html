Strings

Strings are immutable sequences of one-character strings

Sequential Operations

Sequential operations can be performed on strings:

>>> S = 'String'
>>> len(S)
5
>>> S[0]
'S'
>>> S[4]
'g'

Positive indices count from the left, and negative indices count from the left:

>>> S = 'String'
>>> S[-1]
'g'
>>> S[len(S)-1]
'g'
>>> S[-2]
'n'

Slicing is a way to extract a column from a sequence (in this case string). Their general form X[I:J] means extract from offset I up to but not including offset J:

>>> S[:4] # 0 up to 4
'Stri'
>>> S[:-3] # everything except for the last 3
'Str'

A third index can be used to provide a step or a stride:

>>> S[0:4:2]
'Sr'

String repetition is achieved with a * operator:

>>> S * 3
'StringStringString'

The string 

find
method is a basic substring operation which returns the index, or -1 if not found.

The string 

replace
method will replace an occurrence of a substring with another.

>>> S.replace('rin', 'nir')
Stnirg

Split on a delimiter into a list of substrings:

>>> S = '1, 2, 3'
['1',' 2',' 3']

Character code conversions can be performed:

>>> ord('m')
109
>>> chr(109)
'm'

Case conversions:

>>> S = 'String'
>>> S.lower()
'string'
>>> S.upper()
'STRING'

Content tests:

>>> S.isalpha()
True
>>> S.isdigit()
False

Remove whitespace on right side:

>>> S = 'String\n'
>>> S
'String\n'  # repr 
>>> print S
String

>>> S = S.rstrip()
>>> S
'String'
>>> print S
String

A list can be converted into a string with a delimiter like this:

>>> '|'.join(['one','two','three'])
'one|two|three'

Strings can be parsed into a list:

>>> 'one|two|three'.split('|')
['one', 'two', 'three']

Formatting expression:

>>> 'Now %s the %s' % ('is', 'time')
'Now is the time'

Formatting method:

>>> 'Now {0} the {1}'.format('is', 'time')
'Now is the time'

Dictionary-based String Formatting

String formatting can be performed using a dictionary:

>>> "%(a)d %(b)s" % {"a":0, "b":"zero"}
'0 zero'

String formatting can be performed using the vars() built-in function:

>>> a = 0
>>> b = 'zero'
>>> "%(a)d %(b)s" % vars()
'0 zero'

Generic operations that span multiple types show up as built-in functions or expressions, e.g., len(S), S[0]. Type-specific operations are method calls, e.g., S.upper()

To show the available attributes for a type:

>>> dir(S)

To show further information about a specific method:

>>> help(S.replace)
replace(...)
    S.replace(old, new[, count]) -> string
    
    Return a copy of string S with all occurrences of substring
    old replaced by new.  If the optional argument count is
    given, only the first count occurrences are replaced.
