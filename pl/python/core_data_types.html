Python is dynamically typed, and strongly typed.

None is commonly used for initializing names and objects.

A type object is returned by the type built-in function:

>>> type(L)

Booleans

Booleans are integers 1 and 0 with custom display logic shown as True and False

A nonzero number or nonempty object is True

Zero numbers, empty objects, and None are False

Boolean comparisons are done through 'and', 'or', and 'not'

In conditional statements, the object is returned, not the value of True or False

>>> 1 or 2, 2 or 1
(1, 2)
>>> [] or 2
2
>>> [] or {}
{}
>>> 1 and 2, 2 and 1
(2, 1)
>>> [] and {}
[]
>>> 2 and []
[]
